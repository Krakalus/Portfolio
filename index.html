TrustNet: Multi-Dimensional Trust for Misinformation Resistance

Apart Research Defensive Acceleration Hackathon • Nov 2025

  • Co-authored and contributed to implementation of decentralized reputation system modeling trust as weighted vector:
    Trust(a) = 0.25·X(a) [Credentials] + 0.35·P(a) [Vouching] + 0.15·Tₜ(a) [Age] + 0.25·G(a) [EigenTrust]

  • Node.js CLI with JSON ledger; open-source prototype.
